SECTION 1 — Context block

You are analysing a test log from the JioPC Automated Testing Agent.
The log is in JSON Lines format — one JSON object per line.
Each line represents one test result with these fields:
- timestamp: ISO 8601 UTC string
- component: A, B, or C
- test_name: Name of the app or URL tested
- result: The outcome of the test
- duration_ms: Time taken for the test
- detail: A short explanation of the result

The last line of the log is a summary block identified by a "summary": true field, containing aggregate counts.

What each component tests:
- Component A = web app URL reachability and UI element presence
- Component B = native app launch health and process monitoring
- Component C = desktop folder and start menu category presence

Valid result values and their meaning:
- PASS: The test passed, everything worked as expected. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"C","test_name":"JioCinema","result":"PASS","duration_ms":100,"detail":".desktop file found in correct category"})
- FAIL: The test failed due to timeout, HTTP error, missing UI element, or process not found. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"A","test_name":"Google","result":"FAIL","duration_ms":8000,"detail":"Missing elements: input[name=q]"})
- BLOCKED: The web page loaded but showed a bot/Cloudflare challenge. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"A","test_name":"JioSaavn","result":"BLOCKED","duration_ms":3000,"detail":"Bot detection page detected"})
- DEGRADED: The app launched but crashed before the health check completed. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"B","test_name":"JioMeet","result":"DEGRADED","duration_ms":5000,"detail":"Process crashed after 2 seconds"})
- MISSING: The .desktop file was not found anywhere on the system. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"C","test_name":"JioCloud","result":"MISSING","duration_ms":50,"detail":"File not found"})
- MISPLACED: The .desktop file exists but is in the wrong folder or category. (Example: {"timestamp":"2026-06-22T10:00:00Z","component":"C","test_name":"JioSaavn","result":"MISPLACED","duration_ms":50,"detail":"Found in wrong category"})

CRITICAL DISTINCTION: BLOCKED means a Jio URL showed a bot/Cloudflare challenge — this is EXPECTED and NOT a failure. Do not count BLOCKED results as failures in your analysis.
